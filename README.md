## REST сервис для работы над задачами
Выполняет CRUD операции над задачами. Получение списка сущностей реализовано с учётом пагинации.
### Пагинация
Пагинация основывается на 3 основных параметрах, таких как:
- **page** — номер странички (начиная с 0), default = 0
- **size** — размер запрашиваемой странички, default = 10
- **offset** — отступ, default = 0
### Настройки
##### Профили Spring
``-Dspring.profiles.active=dataBase`` устанавливает профили для Spring  
Значение по умолчанию **inMemory**

Доступные значения:
- inMemory — сущности хранятся в памяти
- dataBase — сущности хранятся в базе данных
##### Порт
``-Dserver.port=8081`` устанавливает порт на котором будет развёрнуто приложение  
Значение по умолчанию **8080**
